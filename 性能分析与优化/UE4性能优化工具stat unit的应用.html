UE4 性能优化工具 stat unit 的应用

1. stat unit 工具使用说明

1.1 如何启用

在游戏运行时打开控制台输入：

stat unit

1.2 显示指标说明

指标含义说明
Frame总帧时间当前帧总耗时（ms）
GameGame Thread游戏逻辑线程耗时
DrawRender Thread渲染线程耗时
GPUGPU 耗时GPU 渲染耗时
RHITRHI Thread渲染硬件接口线程
DynRes动态分辨率动态分辨率百分比
FPS ≈ 1000 / Frame(ms)

1.3 如何判断瓶颈

判断规则：
  1. Frame ≈ Game > Draw, GPU → Game Thread 瓶颈
  2. Frame ≈ Draw > Game, GPU → Render Thread 瓶颈
  3. Frame ≈ GPU > Game, Draw → GPU 瓶颈
  4. Frame 远大于三者最大值 → 线程同步/IO/VSync
Frame ≈ max(Game, Draw, GPU)，而非三者之和。

1.4 使用示例

// 场景1: Game Thread 瓶颈
Frame: 45.2ms  Game: 42.1ms  Draw: 8.3ms  GPU: 12.5ms → stat game

// 场景2: Render Thread 瓶颈
Frame: 38.5ms  Game: 6.2ms  Draw: 35.1ms  GPU: 15.3ms → stat scenerendering

// 场景3: GPU 瓶颈
Frame: 50.0ms  Game: 8.1ms  Draw: 7.5ms  GPU: 48.3ms → profilegpu

1.7 注意事项

  1. 编辑器下数据包含编辑器开销，真机更准确
  2. 建议 Development 构建下分析
  3. VSync 可能掩盖真实性能
  4. Frame ≈ max(Game, Draw, GPU)

3. ProfileGPU 使用

profilegpu 是单帧快照，不实时。

  • MobileSceneRender 耗时最高是正常现象
  • 需展开子项查看具体阶段
  • 总耗时与 stat unit 差距大说明没抓到卡顿帧

5. Profiler 分析实战

FQueuedThread::Run.WaitForWork 不是瓶颈！
线程空闲等待，其他地方卡住导致整帧拉长。

6. GameThread 深度分析

GameThread
└── FrameTime ~775ms
    └── World Tick Time ~774ms
        └── LetsGoGameInstance.ReceiveTick ~771ms
            └── LuaGameSE ~229ms
                ├── SendMessageToApp ~129ms ⭐
                │   └── AsyncProtoMessage ~86ms
                ├── CollectPisLogOutput ~42ms ⭐
                ├── CollectXULogOutput ~64ms ⭐
                ├── ExecOnSDKMessage ~19ms
                └── LuaAllocator ~0.05ms

瓶颈排名

排名函数耗时优先级
🥇SendMessageToApp~129ms高
🥈CollectPis/XULogOutput~106ms高
🥉OnSDKMessage~19ms中
4FMsgLog~12+ms低

优化建议

  1. 排查 SendMessageToApp 调用频率，合并消息
  2. Release 环境关闭日志
  3. 检查 ReceiveTick 不必要逻辑
  4. 利用 UpdatePixUITickIntervalSec

关键代码

function PandoraManager:SendMessageToApp(appId, msg)
    if appId == nil then return -1 end
    if type(appId) == "number" then appId = tostring(appId) end
    appId = self.AppId:Trans2EditorAppId(appId)
    local result = UE4.UMoeGameletManager.SendMessageToApp(appId, msg)
    return result
end

function PandoraManager:UpdatePixUITickIntervalSec(intervalSec)
    if m_pixui_tick_interval == intervalSec then return end
    m_pixui_tick_interval = intervalSec
    UE4.UMoeGameletManager.SetTickIntervalSec(intervalSec)
end
